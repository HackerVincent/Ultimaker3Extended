# Ultimaker 3 Extended
Ultimaker 3 Extended Hardware Files

In this repository, you will find the documentation and hardware designs for all parts of the Ultimaker 3 Extended. The bill of materials (BOM) directory lists all parts, with part numbers, quantities and raw materials. Print core AA and print core BB have a separate BOM.

The electronics files can be found in the PCB files directory.
If you are missing a part or information feel free to file an issue on GitHub.
